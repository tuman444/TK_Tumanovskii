## 1. Постановка задачи
Разработать программное обеспечение для вычисления максимума или минимума из трёх вещественных чисел.
Требования:
- Реализация интерфейса на WPF.
- Разработка тестовых сценариев по шаблону.
- Написание автоматизированных Unit-тестов.
- Использование системы контроля версий Git.
- Добавление XAML-комментариев.

3. Реализация приложения
**Интерфейс (UI)**
Дизайн приложения выполнен строго по макету:
- Три поля ввода (TextBox) для вещественных чисел.
- Группа переключателей (RadioButton) для выбора режима: максимум или минимум.
- Кнопка вычисления.
- Текстовая строка для вывода результата.

Архитектура кода:
Для обеспечения высокого качества кода логика вычислений вынесена в отдельный класс Calculator, что позволило реализовать максимальное количество автоматизированных тестов.
